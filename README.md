# A console game: Hangman in Java.
Features:
1. Word masking:
Automatically generates a hidden word from "_" characters at the start of the game.
2. User input processing:
Uses the Scanner to obtain letters from the player in real time.
3. Lives and defeat conditions system:
Limited number of attempts (6 lives) that decrease with each mistake.
4. Dynamic progress update:
Letters are revealed in the mask immediately after the player guesses them correctly.
5. Victory condition checking:
Automatic checking: the game ends when there are no empty spaces left in the mask.

Important: The game only reads the first character of the input. Entering multiple letters at once can result in the loss of a life.

# Консильное игра: Отгадай слово на Java.
Функции: 
1.Маскировка слова:
Автоматическое создание скрытого слова из символов "_" в начале игры.
2.Обработка пользовательского ввода:
Использование Scanner для получения букв от игрока в реальном времени.
3.Система «жизней» и условий поражения:
Ограниченное количество попыток (6 жизней) с уменьшением при каждой ошибке.
4.Динамическое обновление прогресса:
Буквы открываются в маске сразу после того, как игрок их угадал.
5.Проверка условий победы:
Автоматическая проверка: игра заканчивается, когда в маске не остается пустых мест.

Важно: Игра считывает только первый символ ввода. Ввод нескольких букв за раз может привести к потере жизни.
